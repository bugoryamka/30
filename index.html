Неделя 1

Building Blocks + Assets Workflow

Мануал: Building Blocks, Working in Unity — Assets
День 1

Интерфейс Unity

0/6
  • Установка Unity Hub, выбор LTS-версии, установка URP-шаблона
  • Изучить окна редактора: Scene, Game, Inspector, Hierarchy, Project, Console
  • Навигация в Scene View: орбита, зум, flythrough (WASD + ПКМ)
  • Горячие клавиши: Q/W/E/R/T, F (фокус), Ctrl+D (дупликат)
  • Прочитать мануал: Unity Overview → The Editor
  • Практика: создай URP-проект, расставь примитивы, освойся с навигацией
День 2

GameObjects и Components

0/6
  • Что такое GameObject — пустой контейнер для компонентов
  • Component-архитектура: каждый компонент = поведение
  • Transform: Position, Rotation, Scale (локальные vs мировые координаты)
  • Основные компоненты: MeshFilter, MeshRenderer, Collider, Rigidbody
  • Иерархия объектов: Parent-Child, наследование трансформов
  • Практика: построй башню из примитивов с вложенной иерархией, добавь компоненты вручную
День 3

Prefabs, Tags, Layers, Scenes

0/6
  • Prefabs: создание, изменение, Override, Nested Prefabs, Prefab Variants
  • Tags: назначение, стандартные теги, кастомные теги
  • Layers: использование для камеры, физики, освещения
  • Scenes: создание, загрузка, мульти-сцена
  • Static Flags: Batching Static, Navigation Static, Occluder/Occludee
  • Практика: сделай 5 префабов (лампа, стул, стена, дверь, ящик), назначь теги и слои
День 4

Импорт 3D-моделей

0/7
  • Поддерживаемые форматы: FBX (основной), OBJ, glTF
  • Вкладка Model: Scale Factor, Mesh Compression, Read/Write, Generate Colliders
  • Normals: Import, Calculate, None. Geometry: Generate Lightmap UVs, Swap UVs
  • Вкладка Rig: Animation Type (Generic, Humanoid, Legacy, None)
  • Вкладка Materials: Extract Materials, remapping
  • Прочитать мануал: Working in Unity → Assets → 3D model import
  • Практика: скачай 5 моделей с Sketchfab/Poly Haven, импортируй и настрой
День 5

Импорт текстур

0/8
  • Форматы: PNG, TGA, PSD, EXR, HDR
  • Texture Type: Default, Normal Map, Sprite, Cookie, Lightmap, Single Channel
  • sRGB: вкл. для цветовых карт (Albedo, Emission), выкл. для data-карт (Normal, Metallic, AO)
  • Max Size: 512 / 1024 / 2048 / 4096 — выбор по назначению
  • Compression: None, Low, Normal, High Quality
  • Mip Maps: зачем нужны, Mip Map Streaming
  • Format per platform: ASTC (мобильные), BC7/DXT (PC)
  • Практика: импортируй полный PBR-сет, настрой каждую карту корректно
День 6

Asset Store, Package Manager, организация

0/5
  • Unity Asset Store: поиск, импорт ассетов
  • Package Manager: Registry-пакеты, Git URL, Local пакеты
  • Изучить рекомендуемую структуру папок проекта (Art, Prefabs, Scenes, UI, VFX)
  • Asset Labels, Search by Type, фильтрация
  • Практика: скачай 2–3 бесплатных пакета, организуй папки, разбери ассеты
День 7

Сборка сцены и повторение

0/5
  • Собери сцену-комнату из импортированных моделей
  • Все модели — через префабы
  • Назначь слои и теги всем объектам
  • Проверь настройки импорта всех ассетов
  • Результат: организованный проект со сценой
Неделя 2

Graphics — Материалы и Shader Graph

Мануал: Graphics → Materials, Shaders, and Textures → Shader Graph
День 8

PBR-материалы: теория

0/5
  • Что такое PBR (Physically Based Rendering), принцип энергосбережения
  • Metallic/Roughness workflow vs Specular/Glossiness workflow
  • Карты: Albedo, Normal, Metallic, Smoothness, AO, Height, Emission — что описывают
  • Диэлектрики vs Металлы: типичные значения
  • Практика: изучи PBR-сеты на Poly Haven, определи тип каждой карты
День 9

PBR-материалы: практика в Unity

0/5
  • URP Lit Shader: Surface Type, Render Face, Alpha Clipping, все слоты текстур
  • Создание материалов, назначение текстур, Tiling, Offset
  • Detail Maps, Secondary Maps
  • Разница: URP Lit, Simple Lit, Unlit, Complex Lit
  • Практика: создай 6 материалов — металл, дерево, камень, стекло, ткань, пластик
День 10

Shader Graph: введение

0/6
  • Зачем нужен Shader Graph — визуальное программирование шейдеров
  • Создание Shader Graph ассета (URP → Lit/Unlit Shader Graph)
  • Интерфейс: Master Stack, Blackboard, Graph Inspector, Main Preview
  • Основные понятия: Node, Port, Edge
  • Первые ноды: Color, Texture2D, Sample Texture 2D, UV, Multiply
  • Практика: создай простой Lit-шейдер: текстура × цвет → Base Color
День 11

Shader Graph: математика и UV

0/5
  • Математические ноды: Add, Multiply, Lerp, Clamp, Remap, Power, Step, Smoothstep
  • UV-ноды: Tiling and Offset, Rotate, Polar Coordinates, Radial Shear
  • Нода Time — создание анимации. Sine, Cosine — периодические эффекты
  • Fresnel Effect — свечение по краям
  • Практика: шейдер анимированного UV-скролла (река) + шейдер с Fresnel
День 12

Shader Graph: продвинутые техники

0/5
  • Normal Map в Shader Graph (Normal Unpack, Normal Strength)
  • Noise ноды: Simple Noise, Gradient Noise, Voronoi
  • Dissolve-эффект: Step + Noise + Alpha Clip
  • Vertex Displacement: смещение вершин (трава, волны)
  • Практика: dissolve-шейдер + шейдер травы с Vertex Displacement
День 13

Shader Graph: прозрачность и спецэффекты

0/4
  • Surface Type: Opaque → Transparent. Alpha Blending vs Alpha Clipping
  • Шейдер голограммы: Fresnel + Emission + Scanlines
  • Шейдер щита/барьера: Fresnel + пульсация + intersection (Scene Depth)
  • Практика: 2 эффектных шейдера — голограмма и энергетический щит
День 14

Shader Graph: Blackboard и Channel Packing

0/5
  • Blackboard: свойства (Float, Color, Texture2D, Vector, Boolean, Keyword)
  • Channel Packing: 3 текстуры → 1 (Metallic=R, AO=G, Smoothness=A)
  • Ноды Split, Combine для работы с каналами
  • Keyword ноды, Sub Graphs, советы по оптимизации
  • Практика: универсальный PBR Shader Graph с упакованными каналами
Неделя 3

Lighting, Post-processing, VFX

Мануал: Graphics → Lighting → Post-processing → Visual Effects
День 15

Освещение: типы и режимы

0/6
  • Типы источников: Directional, Point, Spot, Area (baked only)
  • Режимы: Realtime, Mixed, Baked
  • Настройки: Intensity, Range, Color, Shadow Type, Bias
  • Environment Lighting: Skybox, Gradient, Color
  • Reflection Probes и Light Probes: назначение, расстановка
  • Практика: настрой освещение — Directional + Point/Spot, расставь Probes
День 16

Lightmapping и Baking

0/5
  • Progressive Lightmapper: CPU vs GPU
  • Настройки: Lightmap Resolution, Padding, Max Lightmap Size
  • Static → Contribute GI. Mixed Lighting режимы
  • Light Probes Group: размещение, тетраэдризация
  • Практика: запеки освещение, сравни Realtime vs Baked
День 17

Пост-процессинг: Volume System

0/6
  • URP Volume System: Global Volume vs Local Volume
  • Volume Profile, Volume Override. Camera Post Processing флаг
  • Bloom: Threshold, Intensity, Scatter, Tint
  • Tonemapping: None, Neutral, ACES
  • Color Adjustments: Post Exposure, Contrast, Color Filter, Saturation
  • Практика: добавь Global Volume, настрой Bloom + ACES + Color Adjustments
День 18

Пост-процессинг: все эффекты

0/6
  • Vignette, Film Grain, Chromatic Aberration
  • Depth of Field: Gaussian vs Bokeh
  • Motion Blur и SSAO (Ambient Occlusion)
  • Anti-Aliasing: MSAA, FXAA, SMAA
  • Lens Distortion, Panini Projection
  • Практика: создай 3 Volume Profile (день, ночь, хоррор)
День 19

Particle System: основы

0/5
  • Модуль Main: Duration, Looping, Start Lifetime, Speed, Size, Color, Gravity
  • Модуль Emission: Rate over Time, Bursts
  • Модуль Shape: Sphere, Cone, Box, Circle, Edge, Mesh
  • Модуль Renderer: Billboard, Stretched Billboard, Mesh. Material, Sorting
  • Практика: базовые эффекты — дым, искры, дождь, снег
День 20

Particle System: продвинутые модули

0/6
  • Color/Size/Rotation over Lifetime — кривые и градиенты
  • Velocity over Lifetime, Noise — управление и турбулентность
  • Texture Sheet Animation — анимированные спрайты
  • Sub Emitters — дочерние эмиттеры. Trails — шлейфы
  • Lights, Collision — частицы-источники света, столкновения
  • Практика: костёр (пламя + искры + дым) и магический портал
День 21

Кастомные шейдеры для частиц + мини-проект

0/6
  • Shader Graph для частиц: Unlit + Transparent
  • Нода Vertex Color — получение цвета от Particle System
  • Soft Particles (Scene Depth). Distortion-шейдер
  • Доработай сцену: PBR-материалы + Shader Graph шейдеры
  • Добавь освещение (Baked + Realtime) + Post-processing Volume
  • Результат: визуально законченная диорама с 2–3 VFX
Неделя 4

Animation, UI, Оптимизация

Мануал: Animation → UI → Optimization / Best Practices
День 22

Animation: клипы и Animation Window

0/6
  • Обзор системы анимации Unity. Animation Window
  • Ключевые кадры: Transform, цвет, свойства компонентов
  • Curves: Linear, Constant, Free Smooth
  • Запись анимации в реальном времени (Record mode)
  • Анимация свойств Material и активности GameObject
  • Практика: анимируй вентилятор, дверь, пульсирующий свет
День 23

Animator Controller и State Machine

0/6
  • Компонент Animator, Animator Controller ассет
  • States, Default State. Transitions: условия, Has Exit Time, Duration
  • Parameters: Bool, Trigger, Float, Int
  • Any State, Entry, Exit
  • Layers: Override, Additive. Avatar Mask (основы)
  • Практика: контроллер для двери — Closed → Opening → Open → Closing
День 24

Blend Trees

0/5
  • Зачем Blend Tree: плавное смешивание анимаций
  • 1D Blend Tree: один параметр (Speed: Idle → Walk → Run)
  • 2D Blend Trees: Simple Directional, Freeform Directional, Cartesian
  • Threshold, Compute Thresholds
  • Практика: скачай Mixamo-анимации, настрой 1D и 2D Blend Tree
День 25

Unity UI (uGUI): основы

0/6
  • Canvas: Screen Space Overlay, Screen Space Camera, World Space
  • Canvas Scaler: Constant Pixel Size, Scale With Screen Size
  • RectTransform: Anchors, Pivot, Stretch
  • Элементы: Image, Button, Toggle, Slider, Dropdown, InputField, ScrollRect
  • Layout Groups: Horizontal, Vertical, Grid. Content Size Fitter
  • Практика: собери HUD — полоска HP, счётчик очков, 3 кнопки
День 26

TextMeshPro + продвинутый UI

0/7
  • TMP Essentials — импорт. Font Asset Creator
  • Rich Text: <color>, <size>, <b>, <sprite>
  • Material Presets: Outline, Shadow, Glow, Bevel. TMP в World Space
  • Mask, RectMask2D — обрезка содержимого
  • Scroll Rect + Content Size Fitter → прокручиваемый список
  • UI-анимация через Animator. Event System, Raycast Target
  • Практика: экран инвентаря со скроллом, TMP-заголовки, анимация панели
День 27

Профилирование и диагностика

0/6
  • Profiler Window: CPU, GPU, Rendering, Memory
  • Frame Debugger: пошаговый просмотр draw calls
  • Stats Window: Batches, Tris, Verts, SetPass Calls
  • Rendering Debugger (URP): Wireframe, Overdraw, Mip Maps
  • Draw Call, Batch, Overdraw, Fill Rate. CPU-bound vs GPU-bound
  • Практика: запиши Profiler, найди тяжёлые draw calls в Frame Debugger
День 28

Оптимизация материалов и текстур

0/7
  • Выбор размера текстур по назначению объекта (512–4096)
  • Compression per platform: ASTC (моб.), BC7/DXT5 (PC)
  • Channel Packing: Metallic(R) + AO(G) + Smoothness(A) → 1 текстура
  • Texture Atlasing. Отключение Read/Write. Sprite Atlas для UI
  • Минимизация Shader Variants / Keywords
  • SRP Batcher, GPU Instancing, Material Property Blocks
  • Практика: оптимизируй текстуры — упакуй каналы, подбери размеры
День 29

Оптимизация сцены и рендеринга

0/8
  • Batching: Static, Dynamic, SRP Batcher — приоритет
  • LOD Group: Level of Detail — настройка порогов
  • Occlusion Culling: Bake, Visualization
  • Camera: Far Clip Plane, Layer Culling Distances
  • Свет: ограничение Range, отключение теней мелких источников
  • URP Asset: Shadow Resolution, Cascade Count, HDR, Render Scale
  • Quality Settings: уровни (Low, Medium, High). Culling партиклов, UI Canvas
  • Практика: добавь LOD, Occlusion Culling, сравни Stats до/после
День 30

Финальный проект

0/10
  • Организованный проект: префабы, слои, теги, структура папок
  • Корректно настроенные импортированные модели и текстуры
  • 5+ PBR-материалов (металл, дерево, камень, стекло, ткань)
  • 2+ кастомных Shader Graph шейдера (dissolve, голограмма, вода и т.д.)
  • Освещение: Baked + Realtime, Light/Reflection Probes
  • Global Volume с Bloom, Tonemapping, Color Grading, DoF
  • 2+ VFX на Particle System (огонь, портал, пыль и т.д.)
  • Анимированный объект/персонаж с Animator + Blend Tree
  • HUD с uGUI + TextMeshPro (HP, текст, кнопки)
  • Оптимизация: LOD, Occlusion Culling, батчинг. Скриншоты/видео для портфолио
