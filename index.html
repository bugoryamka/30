Неделя 1

Building Blocks + Assets Workflow

Мануал: Building Blocks, Working in Unity -- Assets
День 1

Интерфейс Unity

0/6
  • Установка Unity Hub, выбор LTS-версии, установка URP-шаблона
  • Изучить окна редактора: Scene, Game, Inspector, Hierarchy, Project, Console
  • Навигация в Scene View: орбита, зум, flythrough (WASD + ПКМ)
  • Горячие клавиши: Q/W/E/R/T, F (фокус), Ctrl+D (дупликат)
  • Прочитать мануал: Unity Overview -- The Editor
  • Практика: создай URP-проект, расставь примитивы, освойся с навигацией
День 2

GameObjects и Components

0/6
  • Что такое GameObject -- пустой контейнер для компонентов
  • Component-архитектура: каждый компонент = поведение
  • Transform: Position, Rotation, Scale (локальные vs мировые)
  • Основные компоненты: MeshFilter, MeshRenderer, Collider, Rigidbody
  • Иерархия объектов: Parent-Child, наследование трансформов
  • Практика: построй башню из примитивов с вложенной иерархией
День 3

Prefabs, Tags, Layers, Scenes

0/6
  • Prefabs: создание, Override, Nested Prefabs, Prefab Variants
  • Tags: стандартные и кастомные теги
  • Layers: использование для камеры, физики, освещения
  • Scenes: создание, загрузка, мульти-сцена
  • Static Flags: Batching Static, Navigation Static, Occluder/Occludee
  • Практика: сделай 5 префабов, назначь теги и слои
День 4

Импорт 3D-моделей

0/7
  • Форматы: FBX (основной), OBJ, glTF
  • Вкладка Model: Scale Factor, Mesh Compression, Read/Write, Generate Colliders
  • Normals: Import, Calculate, None. Generate Lightmap UVs
  • Вкладка Rig: Animation Type (Generic, Humanoid, Legacy, None)
  • Вкладка Materials: Extract Materials, remapping
  • Мануал: Working in Unity -- Assets -- 3D model import
  • Практика: скачай 5 моделей, импортируй и настрой каждую вкладку
День 5

Импорт текстур

0/8
  • Форматы: PNG, TGA, PSD, EXR, HDR
  • Texture Type: Default, Normal Map, Sprite, Cookie, Single Channel
  • sRGB: вкл. для Albedo/Emission, выкл. для Normal/Metallic/AO
  • Max Size: 512 / 1024 / 2048 / 4096 -- выбор по назначению
  • Compression: None, Low, Normal, High Quality
  • Mip Maps: зачем нужны, Mip Map Streaming
  • Format per platform: ASTC (мобильные), BC7/DXT (PC)
  • Практика: импортируй полный PBR-сет, настрой каждую карту
День 6

Asset Store, Package Manager, организация

0/5
  • Unity Asset Store: поиск, импорт ассетов
  • Package Manager: Registry-пакеты, Git URL, Local
  • Структура папок проекта: Art, Prefabs, Scenes, UI, VFX
  • Asset Labels, Search by Type, фильтрация
  • Практика: скачай 2-3 пакета, организуй папки
День 7

Сборка сцены и повторение

0/5
  • Собери сцену-комнату из импортированных моделей
  • Все модели -- через префабы
  • Назначь слои и теги всем объектам
  • Проверь настройки импорта всех ассетов
  • Результат: организованный проект со сценой

Мануал: Graphics -- Materials, Shaders, and Textures -- Shader Graph
День 8

PBR-материалы: теория

0/5
  • Что такое PBR, принцип энергосбережения
  • Metallic/Roughness vs Specular/Glossiness workflow
  • Карты: Albedo, Normal, Metallic, Smoothness, AO, Height, Emission
  • Диэлектрики vs Металлы: типичные значения
  • Практика: изучи PBR-сеты на Poly Haven
День 9

PBR-материалы: практика в Unity

0/5
  • URP Lit Shader: Surface Type, Render Face, Alpha Clipping
  • Создание материалов, назначение текстур, Tiling, Offset
  • Detail Maps, Secondary Maps
  • Разница: URP Lit, Simple Lit, Unlit, Complex Lit
  • Практика: создай 6 материалов -- металл, дерево, камень, стекло, ткань, пластик
День 10

Shader Graph: введение

0/6
  • Зачем нужен Shader Graph -- визуальное программирование шейдеров
  • Создание ассета: URP Lit/Unlit Shader Graph
  • Интерфейс: Master Stack, Blackboard, Graph Inspector
  • Основные понятия: Node, Port, Edge
  • Первые ноды: Color, Texture2D, Sample Texture 2D, UV, Multiply
  • Практика: простой Lit-шейдер: текстура x цвет в Base Color
День 11

0/5
  • Математические ноды: Add, Multiply, Lerp, Clamp, Remap, Power, Step
  • UV-ноды: Tiling and Offset, Rotate, Polar Coordinates
  • Нода Time, Sine, Cosine -- периодические эффекты
  • Fresnel Effect -- свечение по краям
  • Практика: анимированный UV-скролл + шейдер с Fresnel
День 12

Shader Graph: продвинутые техники

0/5
  • Normal Map в Shader Graph: Normal Unpack, Normal Strength
  • Noise ноды: Simple Noise, Gradient Noise, Voronoi
  • Dissolve-эффект: Step + Noise + Alpha Clip
  • Vertex Displacement: смещение вершин (трава, волны)
  • Практика: dissolve-шейдер + шейдер травы
День 13

Shader Graph: прозрачность и спецэффекты

0/4
  • Surface Type: Opaque / Transparent. Alpha Blending vs Clipping
  • Шейдер голограммы: Fresnel + Emission + Scanlines
  • Шейдер щита: Fresnel + пульсация + Scene Depth
  • Практика: 2 шейдера -- голограмма и энергетический щит
День 14

Shader Graph: Blackboard и Channel Packing

0/5
  • Blackboard: Float, Color, Texture2D, Vector, Boolean, Keyword
  • Channel Packing: Metallic=R, AO=G, Smoothness=A в одну текстуру
  • Ноды Split, Combine
  • Keyword ноды, Sub Graphs, оптимизация
